Шифрование данных в Python без херни

Установить bazel через home brew

brew tap bazelbuild/tap
brew install bazelbuild/tap/bazel

Собрать бинарник

cd python
bazel build ...

Если вы используете zsh вместо bash, то три точки нужно обрамить кавычками “…”
